## Язык третый

Учить английский язык очень легко. Всё знают этот язык. А также, если в твоей профессии, основные ресурси на английском, хочешь или не хочешь, ты должен учить этот язык. Но учить язык как русский труднее. Очевидно число студентов русского языка мальшее. И очень труднее найти люди кто хотят изучать с тобой.

Мир языков человеков очень похож на мир языков программирования. Я думаю что учить новый язык это как учить новый язык программирования. Чтобы хорошо программиравать в твоём новыи языке нужно очень практиковать. Нужно создать много приложений и написать длинные кода.

Мой основный язык программирования был Python долгое время. Python это как английский. Все программисты которые я знаю, имеют программировать на Python. Но сейчас мой основной язык это Rust. Rust было мой любимый язык почти шест лет! 

Rust это более строгий язык чем Python. Использовать Rust сложннее и иногдо может быть даже запутанно! Но это того стоит! По моему мнению, исходные коди на Rust так красивые. Может быть другие думают что я безумный и с ума сошёл, но серьёзно сказать, абсолутно возможно что в начале проекта это легко строить свой программ на простом языке как Python, но преимущество исползовать язык как Rust только будет показать себя когда проект стал большом. Разработать большой проект написано на Rust слышком легкее чем проект написано на Python.

Я буду создать все мои будущие проекты на Rust. Потому что я хочу стать хорошым в этом языке каждый день! А сейчас я могу сказат: русский как Rust для меня! Чтобы написать по-русски ты должен обдумывать каждое слово. Это основная причина почему я открыл этот вебсайт: чтовы практиковать русский как я пишу открытый исходный код.

## Новые слова

- похож на - similar to
- долгое время - for a long time
- строгий - strict
- это того стоит - it's worth it
- серьёзно - seriously
- преимущество - beneft
- Разработать - to develop
- будущий - future
- Обдумывать - to think over
- открытый исходный код - open source code

## Ответ ChatGPT

* **Всё знают этот язык** → *Все знают этот язык* (“всё” = everything)
* **основные ресурси** → *основные ресурсы*
* **труднее найти люди** → *труднее найти людей*
* **кто хотят** → *которые хотят*
* **языков человеков** → *человеческих языков*
* **новый язык это как учить новый язык программирования** → better: *учить новый язык — это как учить новый язык программирования*
* **нужно очень практиковать** → *нужно много практиковаться*
* **длинные кода** → *длинный код / много кода*
* **имеют программировать** → *умеют программировать / программировали*
* **Rust было мой любимый язык** → *Rust был моим любимым языком*
* **шест лет** → *шесть лет*
* **сложннее** → *сложнее*
* **иногдо** → *иногда*
* **исходные коди** → *исходные коды / исходный код*
* **так красивые** → *такие красивые*
* **серьёзно сказать** → *серьёзно говоря*
* **строить свой программ** → *создавать свою программу*
* **исползовать** → *использовать*
* **будет показать себя** → *проявится / покажет себя*
* **когда проект стал большом** → *когда проект станет большим*
* **Разработать большой проект написано на Rust** → *Разрабатывать большой проект, написанный на Rust*
* **слышком** → *слишком*
* **легкее** → *легче*
* **проект написано на Python** → *проект, написанный на Python*
* **Я буду создать** → *Я буду создавать / Я создам*
* **хорошым** → *хорошим*
* **я могу сказат** → *я могу сказать*
* **ты должен обдумывать** → *ты должен обдумывать* is understandable, but more natural: *приходится обдумывать*
* **чтовы** → *чтобы*

## Кейван на русском!

Добро пожаловать в моём вебсайте! Здесь вы можете прочитать примечание я написал для практики русского языка!

Почему я пишу публично? Потому что я думаю что когда я могу что-то создать, на пример приложениу, лучше тоже покожать твою работу другим! Это дает тебе много мотивации. Я этого понял когда я началься поделться исходный код моей приложения на интернете на гитхобе. Это чувствует отлично!

Я разработал метод с котором я буду поделиться мои примечания в этом блоге:

1. Сначала я решу: о чём я хочу написать?
2. Потом я начинаюсь писать только с моими текущими знаниями. Я не использую инструменти как ЧатДЖПТ или переводчик Google. Мне разрешено только найти новые слова на интернете.
3. Сдова который я нашёл я должен написать под моим постом с переводом на английском.
4. Наконец я могу отправить мой окончателный текст в ChatGPT и попросить его исправить мои ошибки.
5. Ответ этого также дольжно бить написано под постом.

### Новые слова:

- примечание = note
- покожать = to show
- другим = to others
- поделться = to share
- исходный код = source code
- текущий - current
- инструменти - tools
- мне разешено - i'm only allowed
- окончателный - final
- попросить - to ask to do smth
- исправить - to correct

### Ответ ChatGPT

* **в моём вебсайте** → *на моём сайте / на моём вебсайте*
* **прочитать примечание** → *прочитать заметку*
* **я думаю что** → *я думаю, что*
* **на пример** → *например*
* **приложениу** → *приложение*
* **лучше тоже покожать** → *лучше тоже показать*
* **твою работу** → *свою работу*
* **Я этого понял** → *Я это понял*
* **началься поделться** → *начал делиться*
* **моей приложения** → *моего приложения*
* **на интернете** → *в интернете*
* **Это чувствует отлично** → *Это ощущается/чувствуется отлично*
* **метод с котором** → *метод, с которым*
* **буду поделиться** → *буду делиться*
* **мои примечания** → *моими заметками*
* **я начинаюсь писать** → *я начинаю писать*
* **с моими текущими знаниями** → *с моими текущими знаниями / используя только текущие знания* (awkward but understandable)
* **инструменти** → *инструменты*
* **мне разрешено только найти** → *мне разрешено только искать*
* **Сдова который** → *Слова, которые*
* **под моим постом** → *под моим постом / под постом* (okay, just repetitive)
* **окончателный** → *окончательный*
* **дольжно бить написано** → *должно быть написано*
